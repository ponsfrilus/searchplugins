How to

There are a few things to do in order to use OpenSearch efficiently in your browser.

Add them all

The easiest way to add them to Firefox is to show the search bar in toolbar. To do so, open Firefox's settings, go to Search > Search Bar and choose "Add search bar in toolbar":

Then reload the page; you should see a "+" icon on the magnifying-glass in the search bar. Click on it and add the OpenSearch you want:

Use them with shortcuts

Once again, open Firefox's settings, go to Search > Search Shortcuts: from there you can define a search shortcut for every OpenSearch you've added.

Usage

Now that they are installed, you can switch back to the "Use the address bar for search and navigation". With shortcuts defined, you can use them to select a search engine:
